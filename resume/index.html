Principais Projetos em Dados

Hackathon Pod Academy | Credit Scoring & Data Platform

Desenvolvimento de uma solução de ponta a ponta para Credit Scoring comportamental, premiada como projeto vencedor pela excelência técnica. A implementação utiliza a arquitetura Medallion com governança ativa, integrando contratos de dados e Data Lineage completo. O ecossistema conta com uma engine de processamento vetorial em DuckDB, Feature Store versionada e modelos de Machine Learning serializados, consumidos via painel de analytics e observabilidade em Streamlit. Tudo isso sustentado por uma infraestrutura Cloud-Ready (OCI, Terraform e Docker), focada em portabilidade total e zero lock-in.

Python DuckDB Apache Airflow PostgreSQL Docker OCI Terraform

Kaggle DataHub | Pipeline de Dados Públicos Brasileiros

Pipeline ETL automatizado para extração, processamento e publicação de dados governamentais (DATASUS, SIOPS, INCA, IBGE e CNES) em dataset público no Kaggle. A solução implementa coleta robusta via FTP com suporte a retry e resume por byte offset, utilizando DuckDB para processamento analítico e carga incremental em Data Lake (MinIO/S3), com automação final via Kaggle API para versionamento e detecção de estado do dataset.

Python DuckDB MinIO Pandas Kaggle API boto3
